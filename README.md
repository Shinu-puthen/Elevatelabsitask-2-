# Power BI Sales Dashboard

![Dashboard Screenshot](Screenshot-2025-11-16-191143.jpg)

## Overview
This repository contains a Power BI dashboard for visualizing sales analytics data. The dashboard provides key insights into total revenue, sales quantity, performance trends, and top customers/products across multiple years[attached_image:1].

## Features
- **Revenue overview**: Displays total revenue and total sales quantity[attached_image:1].
- **Revenue by customer**: Detailed breakdown of revenue and sales quantity for individual customers[attached_image:1].
- **Revenue trend analysis**: Line chart showing monthly revenue fluctuations over several years[attached_image:1].
- **Top customers & products**: Bar charts highlighting the top 5 customers and products based on revenue[attached_image:1].
- **Dynamic filters**: Interactive year and month selections for focused data exploration[attached_image:1].

## Tools Used
- Power BI

## Usage
1. Download or clone this repository.
2. Open the Power BI report file (.pbix) in Power BI Desktop.
3. Use filter controls for year and month to review sales trends and analyze business performance.
